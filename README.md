# wanderworld-adventures

A modern and responsive travel website developed using HTML, CSS, and JavaScript. The website showcases international travel destinations, tour packages, travel galleries, and an online booking form.

---

## 📌 Project Overview

WanderWorld Adventures is a travel agency website designed to help users explore popular international destinations and book exciting tour packages. The website features a clean user interface, responsive design, destination galleries, and form validation using JavaScript.

---

## 🚀 Features

* Responsive Navigation Bar
* Full-Screen Hero Banner
* Popular Destinations Section
* Adventure Travel Section
* International Tour Packages
* Travel Gallery
* Contact & Booking Form
* JavaScript Form Validation
* Mobile-Friendly Design
* SEO Meta Tags
* Hover Effects and Modern UI
